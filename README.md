# Pipex
This project is about handling pipes.

## Deployment
To deploy this project run
```bash
git clone https://github.com/LolinEagle/pipex && cd pipex && make
```

## Arguments
```bash
./pipex <file1> <cmd1> <cmd2> <file2>
```
file1 : Input file

file2 : Output file

cmd1 & cmd2 : shell commands with their parameters
